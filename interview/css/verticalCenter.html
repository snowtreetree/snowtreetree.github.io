# position定位

  • static 正常定位方式，z-index,top等属性不生效
  • relative 相对于自身原本的定位。
  • absolute 元素移出正常文档流，并不为元素预留空间。相对于最近的非static元素进行偏移。
  • fixed 元素移出正常文档流，并不为元素预留空间，相对于屏幕视口。
  • sticky 相对于最近的滚动元素进行偏移。
